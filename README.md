## 🛠️ Installation

```bash

# Install local mongodb or create one on atlas

# Clone the repository
git clone https://github.com/yuvizeref/codelorien.git
cd codelorien

cd backend
# Install dependencies
npm install
# Add .env file using .env.example as a template.
# Start the Backend
npm start


cd ..
cd executer
# Install dependencies
npm install
# Add .env file using .env.example as a template.
# Start the Executer
npm start

cd ..
cd frontend
# Install dependencies
npm install
# Start the Frontend
npm start
```
